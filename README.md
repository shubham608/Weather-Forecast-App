# Weather Forecast Application

This project is a responsive weather forecast application developed using
HTML, Tailwind CSS, and JavaScript. It fetches real-time weather data using
the OpenWeatherMap API.

## Features
- Search weather by city
- Get weather using current location
- 5-day weather forecast
- Recent city dropdown
- Temperature unit toggle
- Extreme temperature alerts
- Responsive design

## Technologies Used
- HTML5
- Tailwind CSS
- JavaScript (ES6)
- OpenWeatherMap API

## How to Run
1. Clone the repository
2. Add your API key in `api.js`
3. Open `index.html` in browser
